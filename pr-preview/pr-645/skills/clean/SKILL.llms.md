# clean — “drive to clean” (alias for `ardi`)

This is a mnemonic alias for the **ardi** skill — drive a PR/MR to a *clean* review verdict. Read and follow the canonical skill:

→ **[ardi](../../skills/ardi/SKILL.llms.md)**

Don’t confuse this with the other “clean” skills: `clean-branches` (`cb`) prunes git branches, `clean-worktrees` (`cw`) prunes git worktrees, and `tidy` / `simplify` clean up code. This alias is only about driving a PR/MR review to a clean verdict.
